🌍 Complete SEO Guide for Next.js (Vercel Hosting)
🔑 1. Understanding SEO

SEO (Search Engine Optimization) means making your website understandable by Google (crawlers) and attractive for users.
Google uses:

Crawling → Googlebot scans your pages.

Indexing → Pages are stored in Google’s database.

Ranking → Pages are shown based on relevance, speed, and quality.

➡️ Our goal is to help crawlers index your site correctly and make your site fast & useful.

📄 2. Meta Tags (Title & Description)

Google shows your title and description in search results.

Why? Helps Google & users know what your page is about.

How? Add unique <title> and <meta name="description"> per page.

✅ Example in app/page.tsx:

import Head from "next/head";

export default function Home() {
  return (
    <>
      <Head>
        <title>Buy Designer Dresses | MyBrand</title>
        <meta name="description" content="Rent and buy designer dresses at affordable prices. Free delivery & easy returns." />
        <meta name="robots" content="index, follow" />
      </Head>
      <main>...</main>
    </>
  );
}


👉 Best practices:

Title: 50–60 chars

Description: 150–160 chars

Must include main keywords

🌐 3. Open Graph & Twitter Meta (Social Sharing)

Why? When users share links on Facebook/Twitter/WhatsApp, you control the preview card.

How?

<meta property="og:title" content="Buy Designer Dresses" />
<meta property="og:description" content="Affordable designer dresses with free delivery." />
<meta property="og:image" content="https://mydomain.com/og-image.jpg" />
<meta property="og:url" content="https://mydomain.com" />
<meta name="twitter:card" content="summary_large_image" />

🗺 4. Robots.txt & Sitemap

Robots.txt tells Google what to crawl.

Sitemap lists all your pages so Google doesn’t miss any.

Setup with next-sitemap:

npm install next-sitemap


Create next-sitemap.config.js:

module.exports = {
  siteUrl: 'https://mydomain.com',
  generateRobotsTxt: true,
}


➡️ After build, you’ll have:

https://mydomain.com/sitemap.xml

https://mydomain.com/robots.txt

👉 Submit sitemap to Google Search Console.

🖼 5. Images SEO

Always add alt text (Google reads it).

Use next/image for lazy loading & optimization.

<Image src="/dress.jpg" alt="Red designer dress for rent" width={600} height={400} />

📊 6. Structured Data (Schema Markup)

Why? Helps Google show rich results (stars ⭐, prices 💰, FAQs).

How? Add JSON-LD.

Example for a product:

<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{
    __html: JSON.stringify({
      "@context": "https://schema.org/",
      "@type": "Product",
      "name": "Red Designer Dress",
      "image": "https://mydomain.com/dress.jpg",
      "description": "Beautiful red designer dress for rent.",
      "brand": "MyBrand",
      "offers": {
        "@type": "Offer",
        "priceCurrency": "USD",
        "price": "49.99",
        "availability": "https://schema.org/InStock"
      }
    }),
  }}
/>

⚡ 7. Performance & Core Web Vitals

Google ranks fast, mobile-friendly sites higher.

✅ Optimize images (next/image)

✅ Use CDN (Vercel has built-in)

✅ Remove unused JavaScript (use dynamic import)

✅ Use Lighthouse / PageSpeed Insights to check performance.

🛠 8. Canonical URLs

Why? Prevents duplicate content issues.

<link rel="canonical" href="https://mydomain.com/page" />

📱 9. Mobile Optimization

Use responsive design (Tailwind, CSS grid/flex).

Test: Google Mobile Friendly Test
.

🔗 10. Internal Linking

Use Next.js <Link> with meaningful anchors.
Bad: "Click here"
Good: "View our Designer Dresses"
